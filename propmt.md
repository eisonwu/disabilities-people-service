# 前文

这是一套 IoT 项目的后台管理系统，用户是残联，其用于管理、服务残疾人与其亲友

# 系统功能

在这里设计一下界面

1. 首页：大体统计视图
2. 档案管理：管理残疾人与其关联亲友的数据
    * 档案列表：显示残疾人档案列表，能够增删改查。详情包含了亲友数据
    * 设备列表：显示已经绑定的设备，并带有最后接收的状态（在线状态、电量、位置等等），能够增删改查
3. 数据监查
    * 监测视图：地图视图，显示管辖区内所有设备；包含筛选功能，如异常设备点等；点击该点能够显示基本信息
    * 设备异常列表
    * 健康异常列表

4. 预警处理
    * 求助列表：接收终端设备推送的越界、心率异常、一键求助等预警信息，支持分级提醒。有筛选功能、派送操作等
    * 工单列表：派送的工单列表，展示处理过程或状态等

5. 统计分析
    * 统计报表：可按时间段展示出报表，有导出 Excel 按钮等，统计内容包含残疾人入档数，预警与服务等等

6. 系统管理
    * 成员管理：管理登录用户列表
